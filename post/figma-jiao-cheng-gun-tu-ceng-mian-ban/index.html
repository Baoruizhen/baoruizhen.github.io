Figma教程丨图层面板

2019-08-22 3 min read

当您在编辑器中时，您将在画布的每一侧看到侧边栏。与工具栏一起，这些侧边栏构成了Figma UI。
Figma UI允许您访问，创建和调整设计元素。
右侧边栏允许您查看和调整任何对象的属性。您还可以访问文件的原型设置并查看任何所选对象的代码。我们将其称为“ 属性”面板。您可以在我们的属性面板文章中了解更多相关信息。
左侧边栏可让您访问文件中包含的图层，资源和页面。我们将其称为图层面板。
在本文中，我们将使用左侧边栏：图层面板让您快速上手。

图层面板顶部有三个标题。您可以使用它们在以下选项之间切换：

  1. 图层面板
  2. 资产组
  3. 页面列表


或者，您可以使用键盘快捷键
MAC系统
图层面板和页面列表：[Option] + [1]
资产组：[Option] + [2]

Windows系统
图层面板和页面列表：[Alt] + [1]
资产面板：[Alt] + [2]

小技巧：您可以使用键盘快捷键切换图层面板的可见性：
macOS：⌘ + Shift +
Windows：Alt + 1 + 2 + 4
您还可以使用“显示/隐藏UI”键盘快捷键切换整个Figma UI的可见性：
macOS：⌘ +
Windows：Ctrl +
*

图层面板

您添加到画布的任何画板，组或对象都将显示在“图层”面板中。
新图层将添加到列表顶部，或者它所在的组或框架的顶部。
“图层”面板将任何子对象嵌套在其父组或框架中。您可以单击“画板”或“组”旁边的箭头以查看任何嵌套的图层：


您可以通过旁边的图标确定图层是框架，组还是对象：

  1. 画板
  2. 组
  3. 组件
  4. 实例
  5. 文本
  6. 形状
  7. 图片
  8. GIF动图

    *小技巧：您可以调整“图层”面板的宽度：
  9. 将光标悬停在侧边栏的右边缘上。将出现双向箭头。
  10. 单击并拖动以调整面板的宽度。
  11. 释放以设置侧边栏宽度。*

选择图层

你可以单击“图层”面板中的“图层”将其选中。
选定的图层将在“图层”面板中显示为蓝色。Figma还将在画布中的图层周围显示一个蓝色边界框。

您还可以通过将鼠标悬停在“图层”面板中的“图层”上来突出显示画布中的图层。

小技巧：您可以启用或禁用“ 悬停时突出显示图层”设置。在菜单中搜索“突出显示”：

或者转到菜单>首选项>悬停时突出显示图层：
*

重新排列图层

您可以更改图层在图层面板中显示的顺序。
这使您可以移动特定的图层到前面或画布的上面，以及移动图层中的一组或者画板。
在“图层”面板中调整图层不会影响其在画布上的位置。选择或分组图层的方式，可能会更改您在画布中查看。
您可以通过以下几种方式重新排序图层：

从鼠标右键菜单：

  1. 选择画布中的图层。
  2. 右键单击图层以打开菜单：
  3. 然后您可以选择：
    往上放一层 macOS： ⌘+ 】 | Windows： Ctrl + 】
    放在最前 macOS： ⌘+⌥+ 】 | Windows： Ctrl + Shift + 】
    向后放一层 macOS： ⌘+【 | Windows： Ctrl + 】
    放在最后退macOS： ⌘ + ⌥ + 【 | Windows： Ctrl + Shift + 】

从图层面板：

单击并拖动图层以更改其在层次结构中的位置。

  • 将图层移向列表顶部会将图层置于画布的前面。
  • 沿着列表向下移动图层将图层置于画布中的这些图层的后面。

小技巧：如果您做出改变而不满意，可以使用键盘快捷键撤消操作：
macOS：[⌘] + [Z]
Windows：[Ctrl] + [Z]*

改变图层可见性

您可以在“图层”面板中切换图层的可见性。您仍然可以更改图层位置，或调整隐藏图层的属性。
您可以通过“ 图层”面板切换图层的可见性：

  1. 将鼠标悬停在“图层”面板中的“图层”上，直到出现眼睛和挂锁图标：
  2. 单击眼睛图标以关闭图层：
  3. 画布中将不再显示图层。它也将在“图层”面板中显示为非活动状态。
  4. 要再次显示图层，请单击关闭的眼睛图标：

小技巧：你也可以使用键盘快捷键
macOS：[⌘] + [Shift] + [H]
Windows：[Ctrl] + [Shift] + [H]*

锁定图层

您可以锁定图层以防止意外编辑。
锁定图层时，您将无法与其进行交互，也无法在画布中移动它。它仍然可以更改属性在属性面板中锁定图层的。
如果锁定画板或组，则该画板或组的任何子图层也将被锁定。您可以选择单独解锁任何子图层。
您可以通过几种不同的方式锁定或解锁图层：

使用键盘快捷键：

macOS：[⌘] + [Shift] + [L]
Windows：[Ctrl] + [Shift] + [L]

右键单击图层并选择锁定/解锁：

<图像：锁定/解锁菜单>

从图层面板：

  1. 将鼠标悬停在“图层”面板中的“图层”上，直到出现挂锁图标：
  2. 单击挂锁以锁定图层：
  3. 如果这是一个画板或组，那么该画板/组的任何子项也将被锁定。您可以单击子图层旁边的点来单独解锁它们：
  4. 您将无法再在画布中编辑图层。
  5. 要解锁图层，请再次单击挂锁：

小技巧：你可以一次切换多个图层的图层可见性和锁定。
单击眼睛/挂锁图标，然后在要改变的图层上拖动。
*

资产组

您可以重复使用设计的各个方面。这可以是按钮或图标，也可以是更复杂的UI元素，如导航菜单或状态栏。
您可以创建这些元素的组件，您可以在设计中重复使用这些元素。您对主组件所做的任何更改都将可用于任何相关实例。
您可以从资产面板查看所有组件：

这将来自当前文件的组件以及您有权访问的任何文件或库组合在一起。
您可以通过单击“图层”面板中的“ Assets”标题来打开“资源面板。或者通过使用键盘快捷键：

  • MacOS：[Option] + [2]
  • Windows：[Alt] + [2]

查找组件

您可以使用“ 搜索”查找合适的组件：

这将查找当前文件中的组件以及您有权访问的任何库。
然后，您可以将“资源”面板中的“组件”拖动到画布上。

资产面板中的组件按以下顺序排列：

  1. 您在当前文件中创建的组件。
  2. 文件专用的组件。这包括此文件中未发布到团队库的任何组件。
  3. 您在当前文件中使用的其他文件中的组件。
  4. 任何已启用的团队库中的组件。

小技巧：您可以使用视图图标在网格和列表视图之间切换：

*

团度资源库

如果您在专业团队中，您还可以从资产面板访问团队库：

团队库允许您将文件中的组件发布到库中。然后，您可以创建组件的实例并在其他文件中使用它们。

页面

页面允许您在单个文件中创建单独的画布。
您可以向文件添加的页数没有限制。如果您的扩展超出了文件的最初目的 - 或者您的文件变得越来越大 - 您可能需要考虑使用新文件。
添加页面非常适合：

  • 存储与文件相关的组件和样式
  • 测试您的设计的迭代
  • 保持设计版本分开
  • 针对不同的Web或移动平台进行设计
  • 创建不同的设计原型。

小技巧：您可以通过在单独的页面上创建多个原型来在文件中创建多个原型。在我们的Prototyping 入门文章中了解有关创建Prototype的更多信息。*

浏览页面列表

您可以在图层面板的顶部找到当前页面的名称。
要打开页面列表，请单击当前页面名称旁边的箭头：

这将打开“图层”面板，您可以在其中查看该文件中的页面列表。

然后你可以：

  • 单击+图标将新页面添加到文件中：
  • 单击并拖动页面以更改顺序：
  • 双击页面以更改名称：
  • 右键单击页面以打开“页面”菜单。这将为您提供以下选项：
  1. 复制页面
  2. 删除页面
  3. 重命名页面
  4. 创建一个新页面
  • 单击当前页面旁边的箭头以折叠或展开页面列表：

在页面之间移动图层

您可以在页面之间移动图层。当你想要分解一个大文件时，这很方便。这也允许您保持与这些图层关联的任何评论不变。

  1. 在画布或“图层”面板中选择“对象”，“画板”或“组”。
  2. 右键单击以打开菜单：
  3. 选择菜单顶部的Move to Page：
  4. 选择您要将其移动到的页面：
  5. 对象，框架或组将移动到新页面上的相同坐标。

词汇表

  • 对象是您添加到画布的单个元素。这些可以是形状，矢量网络，文本对象或图像。
  • 组是对象的集合。它们允许您移动和定位对象，就好像它们是单个图层一样。组采用其中包含的对象的组合维度。组没有自己的任何属性。您应用于组的任何属性都将应用于单个对象。
  • 与组一样，画板也是对象的集合，但不是Groups。它们允许您控制对象的位置，对齐和约束。您可以将画板嵌套在其他画板中。了解有关何时在“ 组和画板”文章中使用“组”或“ 画板 ”的详细信息。
  • 子和父是通常用于Web开发的术语。我们使用这些术语来解释Frame（父）与其中的任何对象（子）之间的关系。子项的约束和对齐将始终与其父项相关。
  • 组件是您可以在设计中重复使用的UI元素。这可以是按钮，导航菜单，图库或布局。要重用组件，您可以创建实例。详细了解组件。
