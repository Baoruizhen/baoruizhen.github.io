Figma入门

欢迎来到Figma，这是世界上第一个协作界面设计工具（🐂🍺）。
你可以在一个任何地方使用Figma做UI原型设计，并帮助您更快地将您的想法转化为产品。
Figma是为浏览器构建的，因此您可以在任何平台上（Windows，Mac，Linux和Chromebook）使用它。无需下载或更新。
对于那些喜欢应用体验的用户，可以下载适用于Windows和Mac的Figma桌面应用。或者使用Figma Mirror应用程序将您的设计演示提升到一个新的水平。通过https://www.figma.com/downloads/了解更多信息并下载应用程序

  • 如果您想在浏览器中使用自己的本地字体和Figma，那么您也可以下载Figma的Font Helper应用程序。注意： Linux设备目前不支持使用本地字体。*

开始使用Figma

如果您对设计工具有经验，那么Figma的设计工作台可能看起来很熟悉！但是，可能有一些事情与您之前使用过的其他设计软件有些不同。
以下是一些技巧，让您更好地了解和使用Figma。

  1. 确保您的浏览器受支持。
    最低浏览器要求是：
    Chrome 49+
    Firefox 51+
    Safari 10.1+（Apple - macOS）
    Edge 14+（Windows）
  2. 设置浏览器以使用Figma以最佳方式工作。
  3. 创建您的Figma帐户。
  4. 参观Figma界面。
  5. 查看所有键盘快捷键。
  6. 在Figma建立你的团队。
