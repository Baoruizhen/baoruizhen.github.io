Figma教程丨图层相关

2019-08-23 2 min read

组和画板

组和画板允许您将图层组合在一起，以便可以将它们视为单个图层。

组

默认情况下，组采用其子项的组合维度。

  • 要创建组，请选择一些图层并选择“组/Group”（⌘+ G）。
  • 第一次单击该组中的图层时，将选择整个组。
  • 要选择该组中的单个图层，请双击该项目，或从图层面板中选择该图层。
  • 当组内的图层移动时，组的边界将调整以适合其中的图层。
  • 您可以通过拖动图层面板中的图层来移动和移出该组。
  • 您可以在其他组中嵌套组。
  • 要取消分组，请选择该组并选择“取消组合/UnGroup”（⇧+⌘+ G）。

画板

有时，拥有一个大小由您明确设置的图层很有用。这些被称为画板。

  • 要制作画板，请选择画板工具（快捷键F）并拖出屏幕区域。
  • 要向此组添加图层，只需将其拖入即可。
  • 要从该组中删除图层，只需将它们拖出即可。
  • 选择和移动框架内的项目与组的工作方式完全相同。
  • 要取消分组，请选择该组并选择“取消组合”（⇧+⌘+ G）
  • 画板直接在画布上绘制，我们以特殊方式绘制它们，使它们更易于查看和使用。

画板工具


在“区域工具”的下拉菜单下，您将找到“画板工具”。要向画布添加画板：
选择画板工具（快捷键F或A）
从右侧的“属性”面板中选择“画板预设”，或者直接在“画布”上单击并将“画板”拖动到所需的大小

缩放工具

默认情况下，文件打开时启用移动工具。单击下拉图标（或使用“K”快捷方式），您将找到“缩放”工具。

虽然无法按照数字缩放，但可以拖出边界框的手柄，内部的内容将缩放。
你可以缩放：

  • 画板
  • 组
  • 组件
  • 组件实例
  • 文字对象
    任何模糊和笔画也会缩放。

重命名图层

建立清晰的命名结构或层次结构使得管理文件中的图层变得更加容易。当使用Figma的团队库和/或样式功能时，这一点变得尤为重要。
当您想要快速切换团队库中的相关组件时，这会派上用场。

打开“重命名图层模式”

您可以通过多种不同方式访问“ 重命名图层模式”。

  1. 从画布或图层面板中选择要重命名的图层：
  2. 要切换重命名图层模态， 您可以执行以下操作之一：
  • 右键单击画布中的图层，然后选择“ 重命名图层”：
  • 右键单击面板中的图层，然后选择“ 重命名图层”：
  • 使用键盘快捷键：
    Mac：[ command⌘ ] + [ R ]
    Windows：[ Ctrl ] + [ R ]
  1. 在重命名图层模式将在画板上打开：

重命名图层

如果您只想将所选图层更新为相同名称，则可以将所需名称添加到“ 重命名为” 字段，然后单击“ 重命名”。
但是，您更有可能希望更新每个图层以使名称略有不同，从而可以更好地区分图层面板中的对象。
在重命名字段下方，如上图您将看到三个按钮。单击时，这些按钮将向重命名字段添加一个特殊代码，允许您为每个图层生成略有不同的名称。
点击 Current name 按钮表示以当前图层的名称。
点击 Number↑ 按钮插入一个编号到每个层的名称，以升序排列。
点击 Number↓ 按钮插入一个编号到每个层的名称，在降序排列。

重命名图层名称一部分

在使用重命名时，您可能已经注意到“ 匹配 ”字段。这允许您改变的图层名称的某一部分。
留出空白可以更新整个名称，但您也可以使用此字段删除或更新图层名称的一部分（例如，将“ Icon / Home ” 名称中的“ Icon/ ”部分更改为“ Image/ ”） 。

  1. 选择要重命名的图层。
  2. 在“ Match” 字段中，输入您要改变的名称部分。
  3. 输入您要用该名称替换该部分名称的内容。这只会替换匹配字段中标识的图层名称部分。
  4. 您将在左侧看到更新名称的预览。单击“重命名”按钮以应用。

使用正则表达式（进阶技巧）

如果你熟悉使用正确的表达式，那么你也可以在使用这些重命名图层。如果要更改名称的规则（例如，将“ Icon_003”更改 为“003_Icon”），这将特别方便。

  1. 选择要重命名的图层。
  2. 使用键盘快捷键打开 重命名图层模式：
    Mac：[ command⌘ ] + [ R ]
    Windows：[ Ctrl ] + [ R ]
  3. 在匹配字段中输入“（[a-zA-Z] +）_（\d +）” 。
  4. 在“ 替换”字段中输入“$2_ $1” 。
  5. 您将在左侧看到更新名称的预览。
  6. 单击“ 重命名”按钮以应用。

选择图层和对象

在更新与对象或图层关联的任何属性之前，您需要选择它。

小技巧：要快速选择“画布”中的所有对象，可以使用键盘快捷键[⌘] + [A]（Mac）或[Ctrl] + [A]（Windows）。

选择单个对象

可以通过多种方式选择对象：

从画布

  1. 单击“画布”中的对象。
  2. 对象边缘周围会出现一个蓝色边框，每个角上都有一个操作手柄。

    如果对象嵌套在另一个对象，组或图层中，则可以双击该对象以选择向下一级的图层。然后，您可以重复双击，直到找到正确的选择。
    或者，您可以通过在单击画布中的对象的同时按住 [ ⌘] （Mac）或 [Ctrl]（Windows）键来使用深度选择功能，以直接选择被嵌套的对象。

从图层面板

您还可以从“图层”面板中选择一个图层：

  1. 图层按页面排列在图层面板中，并根据它们的对象类型命名。

通过逻辑和易于识别的名称命名您的图层，将帮助您在“图层”面板中找到它们。查看批量重命名图层文章。

  1. 如果页面上有任何画板或组，则这些组或框架中的对象将嵌套在“图层”面板中的该画板或组中。
  2. 然后，您可以使用“画板”或“组”旁边的箭头来查看任何嵌套对象。
  3. 如果将鼠标悬停在面板中的图层上，蓝色框将突出显示该图层在“画布”上的位置。
  4. 单击对象以选择它。

选择多个对象

您还可以一次选择多个对象或图层。当您想要跨多个层更新属性，批量调整图层大小，移动对象集合时，这非常有用; 或从选择中创建组，画板或组件。

从画布

  1. 在“画布”中选择一个对象。
  2. 选择对象后，按住[Shift]键并单击另一个对象。
  3. 这将允许您选择任意数量的对象。
    在按住shift的同时再次单击一个对象，将从当前选择中删除它。

    您还可以使用选框工具来选择多个对象 ：
  4. 单击并按住画布的空白部分。
  5. 将光标拖动到您要选择的任何对象上。
  6. 蓝色框将指示选择中包含哪些对象。
  7. 要使用选框选择嵌套图层，请按住[⌘]（Mac）或[Ctrl]（Windows）并将选框拖动到您要包含的任何对象上。

从图层面板

这需要稍微不同的方法。

  1. 在“图层”面板中选择一个图层。
  2. 要选择两个图层之间的所有图层，请按住[Shift]键并选择另一个图层：
  3. 要单独选择多个图层，请按住 [⌘] （Mac）或 [Ctrl] （Windows）键并选择另一个图层：

小技巧：您还可以使用“编辑”菜单根据其属性选择多个对象。选择图层后，您还可以选择其他所有具有相同图层的图层：属性，填充，描边，效果，文本属性，字体或其他实例。

选择嵌套对象

如果您正在进行更复杂的设计，则可能会有一堆嵌套在其他组或画板中的对象。
单击属于组或画板的对象时，默认情况下将选择整个画板/组。
我们有几种不同的方法来选择嵌套对象。我们称之为选择图层。

  1. 单击选择图层时按住[⌘]（Mac）或[Ctrl]（Windows）或键。
  2. 为了更加准确，您可以按住[⌘]（Mac）或[Ctrl]（Windows）并右键单击对象以打开“选择图层”迷你菜单。
    这允许您查看光标下的完整图层列表。图层图标和名称将以与图层面板相同的顺序显示，但不表示层次结构。

小技巧：组件和实例中的图层将显示为紫色图标。

  1. 您还可以通过右键单击对象并选择“选择图层”来进入“选择图层”菜单。

使用锁定和隐藏图层

如果隐藏了图层，则它将不会显示在“选择图层”菜单中。必须先取消隐藏图层，然后才能在此菜单中选择该图层。
如果对象或图层被锁定，则无法通过正常的左键单击选择过程进行选择; 但是，可以通过“ 选择图层” 进程选择它。“锁定”图层包含在“选择图层” 菜单中，并带有挂锁图标以指示它们已锁定。
